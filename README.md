# School-Management-System
نظام إدارة المدارس (School Management System)
وصف المشروع
تطبيق ويب ديناميكي متكامل يهدف إلى أتمتة العمليات الإدارية داخل المؤسسات التعليمية، مما يسهل إدارة سجلات الطلاب والمدرسين والفصول الدراسية بفعالية ودقة.
فكرة ال
تقوم الفكرة على توفير لوحة تحكم شاملة للمسؤول (Admin Dashboard) تتيح له تنفيذ العمليات الأساسية (CRUD) على قاعدة البيانات، مع نظام حماية متطور وتوزيع للصلاحيات.
المتطلبات التقنية المحققة
قاعدة البيانات: تحتوي على 5 جداول مترابطة بعلاقات منطقية (Students, Teachers, Classes, Users, Subjects).
الأمان: استخدام الـ Sessions والـ Cookies لإدارة الجلسات، وتطبيق مفاهيم الـ Validation.
العمليات: تطبيق كامل لعمليات الإضافة، العرض، التعديل، والحذف (CRUD).
لتصميم: واجهات متجاوبة ومنظمة باستخدام إطار العمل Bootstrap
طريقة التشغيل
قم بتحميل المشروع (Clone) من الرابط التالي: 
https://github.com/wafiareej-stack/School-Management-System
قم باستيراد ملف قاعدة البيانات school_management.sql عبر phpMyAdmin.  
افتح المتصفح وتوجه إلى الرابط: localhost/School-Management-System.
بيانات تسجيل الدخول التجريبية
الدور (Role)	اسم المستخدم	كلمة المرور	الصلاحيات (Permissions)
مدير النظام (Admin)	admin	admin123	إدارة المدرسين، الطلاب، الفصول، والصلاحيات الكاملة.
مستخدم/موظف (Staff)	user01	user123	عرض البيانات، إضافة طلاب جدد (بدون صلاحية الحذف أو تعديل المدرسين).
